# 🚀 Public API Playground | Explorer

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://public-api-explorer-wm4knfagy-himanshus-projects-84e1edf6.vercel.app/)

A premium, interactive web application for exploring and testing various public API endpoints. This tool provides a clean and elegant interface to fetch real-world data from multiple sources in real-time.

This project was designed, developed, and polished with the assistance of **Antigravity AI**. Leveraged state-of-the-art AI capabilities to create a high-performance, aesthetically pleasing, and user-friendly experience.

---

## ✨ Features

- 🐕 **Dog Finder**: Fetch random dog images from a collection of thousands of breeds.
- 😂 **Joke Generator**: Get your daily dose of laughter with random jokes across different categories.
- 👤 **Random User Profile**: Generate complete, realistic user profiles with avatars, names, emails, and contact details.
- 📄 **JSON Explorer**: Fetch and inspect raw JSON data from various endpoints including Posts, Comments, Albums, and Photos.
- 🎨 **Premium UI/UX**: Built with a sleek, responsive design featuring glassmorphism, smooth animations, and a modern color palette.
- 📋 **One-Click Actions**: Easily copy image URLs and other data to your clipboard.
- 📱 **Mobile Responsive**: Fully optimized for various screen sizes, from mobile devices to large desktop monitors.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure and accessibility-focused layout.
- **CSS3**: Modern styling using variables, flexbox, grid, and glassmorphism effects.
- **Vanilla JavaScript**: Lightweight and fast interactive logic with `fetch` API for asynchronous data loading.
- **Ionicons**: High-quality vector icons for enhanced visual clarity.
- **Google Fonts**: Premium typography using *Inter* and *Outfit*.

## 🌐 APIs Used

- [Dog CEO API](https://dog.ceo/dog-api/)
- [Official Joke API](https://official-joke-api.appspot.com/)
- [Random User Generator](https://randomuser.me/)
- [JSONPlaceholder](https://jsonplaceholder.typicode.com/)

## 🚀 Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your preferred web browser:

```bash
# Clone the repository
git clone https://github.com/Himanshubansal-2008/public-api-explorer.git

# Navigate to the project directory
cd public-api-explorer

# Open index.html in your browser
# (Or use a live server extension in VS Code)
```
